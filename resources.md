Resources
========

The RSGISLib software package has a number of useful segmentation tools - see http://rsgislib.org/rsgislib_segmentation.html
The paper at https://www.mdpi.com/2072-4292/11/6/658 (PDF at https://www.mdpi.com/2072-4292/11/6/658/pdf) describes one of the segmentation algorithms there

There is a description of an end-to-end system using RSGISLib here: https://www.mdpi.com/2072-4292/6/7/6111 (PDF at https://www.mdpi.com/2072-4292/6/7/6111/pdf)

Don't be afraid to download the PDFs and read in detail
